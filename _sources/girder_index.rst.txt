Girder Integration
=======================

Girder_ is a free and open source web-based data management platform developed by Kitware_. Girder is both a standalone application and a platform for building new web services. ``large-image`` has a Girder plugin that can be added to any Girder instance to manage and view large images.

To see an example Girder application with the ``large-image`` plugin, visit `demo.kitware.com <https://demo.kitware.com/histomicstk/#folder/589e030a92ca9a00118a6166>`_.
